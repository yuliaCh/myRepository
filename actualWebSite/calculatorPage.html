Как работает приложение

Программа Калькулятор производит вычесления с данными числового формата, введеными в поля "Число1" и "Число2" при нажатии на одну из кнопок со знаком "+", "-", "*" или "/". Результат вычислений выводится в поле "Результат". Для того чтобы очистить поля приложения необходимо нажать на кнопку "Очистить поля".

Валидация текстовых полей

Вычисления производятся с целыми и дробными (с разделителем в виде точки) числами. В случае когда в поле "Число1" и/или "Число2" вводятся данные не являющияся числом, при нажатии на одну из кнопок со знаком вычисления данное поле очищается и в поле "Результат" выводится сообщение "Ошибка". Также, сообщение об ошибке выводится при делении на ноль. Числа введенные с внешними пробелами допускаются. Пустые поля воспринимаются как поля со значением ноль. Поле "Результат" служит для выведения результатов вычислений и сообщений об ошибке, введение данных в это поле не допускается.
