Calculator

Application Description

How the application works

Calculator application performs calculations with numbers, entered into the "Number 1" and "Number 2" text fields, when one of the buttons with the following math symbols is clicked on: "+", "-", "*" or "/". The result of these calculations is displayed in the "Result" text field. In order to clear the fields, the user needs to click on the "Clear Fields" button.

Text Field Validation

The application works with integers and decimal numbers. In case the user enters a non-numerical character into the "Number 1" or "Number 2" field, the field gets cleared and the "Result" field displays "Error" message upon user's clicking on one of the buttons with a math symbol. The error message also gets displayed when there is an attempt to divide a number by zero. Leading and trailings whitespaces are accepted. Empty text fields are treated as if they contain zero. The "Result" field is a read-only field with a purpose to display the result of calculations and error messages. It is not possible to enter any data into this field.
